# Climate Exercise

This repository provides documentation and examples for working with World Bank climate and economic data APIs.

## API Reference

| API | Base URL |
|-----|----------|
| Indicators API v2 | https://api.worldbank.org/v2/ |
| Climate Data API | https://climateknowledgeportal.worldbank.org/api/data/ |

## Documentation

- [indicators.MD](indicators.MD) - Complete list of World Bank indicator codes and definitions
- [quick-reference.md](quick-reference.md) - Examples and quick reference for querying data

## Getting Started

The World Bank provides two main APIs for accessing climate and economic data:

1. **Indicators API v2**: Access economic, social, and environmental indicators for countries and regions
2. **Climate Data API**: Access climate-related data including temperature, precipitation, and projections

Refer to the [quick-reference.md](quick-reference.md) for practical examples of how to use these APIs.
